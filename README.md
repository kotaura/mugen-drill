# おうち学習ドリル

小学1年生・小学4年生向けのブラウザ学習ドリルです。各ページは単独のHTMLファイルなので、GitHub Pagesなどの静的ホスティングにそのまま置けます。

## 収録ページ

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ |
| `curriculum.html` | 学習カリキュラム早見表 |
| `grade1-japanese.html` | 小学1年生 国語 |
| `grade1-math.html` | 小学1年生 算数 |
| `grade4-japanese.html` | 小学4年生 国語 |
| `grade4-math.html` | 小学4年生 算数 |
| `grade4-science.html` | 小学4年生 理科 |
| `grade4-social-studies.html` | 小学4年生 社会 |

## 使い方

1. `index.html` をブラウザで開きます。
2. 学年・教科を選んで問題を解きます。
3. 「問題を印刷」で問題用紙、「解答を印刷」で解答用紙をA4印刷できます。
4. 「画像で保存」で現在のシートをPNGとして保存できます。

## GitHub Pages で公開する場合

1. このフォルダの中身をGitHubリポジトリにアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、`main` ブランチの `/root` を公開元にします。
4. 表示されたURLを開くと `index.html` がトップページとして表示されます。

## 変更メモ

- ファイル名を日本語から英数字のkebab-caseに整理しました。
- トップページと各ページのリンクを新しいファイル名に更新しました。
- 解答印刷では、固定幅の入力欄に答えを流し込む方式をやめ、折り返し可能な解答ラベルを表示する方式に変更しました。長い語句や別解がある答えも読めるようにしています。
- 問題バリエーションを拡張しました。クイズ形式のページは大きめの問題プールからランダムに一部を出題するため、同じ単元でも開くたびに内容が変わりやすくなっています。
- 4年算数に「大きい数・がい数」「単位の変換」「表とグラフ」「直方体・立方体」を追加しました。

## 注意

- Google Fonts、Tailwind CSS、html2canvasをCDNから読み込んでいます。完全オフライン対応にする場合は、これらの依存関係をローカル化してください。
- ライセンスは未設定です。公開前に必要なライセンスファイルを追加してください。
